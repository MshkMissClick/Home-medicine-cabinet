## Замечания
* Ежедневные уведомления стабильны, но для больших баз данных рекомендуется оптимизация запросов (индексы на `expiry_date`).
* Фронтенд адаптивен; формы валидируют поля, уведомления о ошибках отображаются.
* Рекомендуется добавить JWT-авторизацию для всех эндпоинтов, кроме публичных.
* Нет критических дефектов; покрытие ~85%.
